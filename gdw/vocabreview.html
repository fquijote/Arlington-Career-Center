HTML Vocabulary Review

Image
An image is an exact copy of the contents of a storage device. <img src="location image file" alt="alt string">. This is the command layout used to put images in a webpage.
Path
A list of directories where the operating system looks for executable files if it is unable to find the file in the working directory.
Attribute
A specification that represents a property of an object, file, or element. They are modifiers of html elements with values assigned to names using name="value" inside the open tag of an element. The href attribute is required for hyperlink elements.

Value
The content of a field or variable. It can refer to alphabetic as well as numeric date. It can be virtually any kind of data type like a digit or a single letter.
Data
Information that has been gathered and translated for analysis. It can be a character, including text and digits, pictures, video, or sound.
Metadata
Is information that provides information about other data.
Hypertext
Text displayed on computer and other electronic devices with hyperlinks to other text which the reader can immediately access.
Hyperlink
A reference to data that the reader can directly follow by clicking, tapping, or hovering. It points to a whole document or to a specific element within a document.
Absolute Location
An absolute location uses the entire URL, like : <<http://openbookproject.net/tutorials/getdown/html/index.html>> The cons of using absolute location is that it is too extensive and also makes it difficult to move a site.
Relative Location
It is the best and easy alternative to absolute location. It works when a user has set two files in a same directory which makes it have the same relative location. Can be written as: <a href="lesson7.html">Lesson 7</a>
Table
HTML tables are designed to hold tabular data which is a systematic arrangement by columns and rows.
Table Data
Organized information that is set in an arrangement of rows by significance.
List
A collection of list items. Here is how it should look like:
    <ul>
      <li>apples</li>
        <li>bananas</li>
          <li>cherries</li>
            <li>grapes</li>
            </ul>
Unordered List
It is a list item and is represented as ul where each item is between <li></li> tags. This list is closed with a </ul> tag.
Ordered List
Used for enumerated items which starts with <ol> and closing in </ol>.
Definition List
Used to list words in a certain arrangement with their definitions. An example of the source code: 
     <dl>
       <dt>HTML</dt><dd>Hyper-text Markup Language</dd>
         <dt>WWW</dt><dd>World Wide Web</dd>
           <dt>W3C</dt><dd>World Wide Web Consortium</dd>
             <dt>dl</dt><dd>definition list</dd>
               <dt>dt</dt><dd>definition term</dd>
                 <dt>dd</dt><dd>definition description</dd>
                 </dl>
Inline Element
Occupies only the space bounded by the tags that define the inline element such as <strong></strong>,<b></b>, and <i></i>.
Object Element
These elements are used to insert objects in and between texts to separate sections of writing or just to mainly make the webpage nice to look at. There are three object elements:
<br>,<img>, and <hr>.
Block level elements
Marks up larger sections within documents.
Angle Brackets
Used to to enclose text in various technical contexts.
Stored quantities,characters, or symbols on a computer
Tag
A kind of metadata that helps describe an item and allows it to be found again by browsing or searching.
